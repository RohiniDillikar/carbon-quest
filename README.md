# Carbon Quest - Personal Carbon Footprint Tracker
Carbon Quest is an interactive web application that helps users understand,track, and reduce their personal carbon footprint.
The platform combines visual dashboards, eco-friendly tips, rewards, badges, and a leaderboard to make sustainability engaging and motivating.

# Problem Statement
Climate change is accelerating due to increasing carbon emissions from daily human activities such as transportation, electricity usage, and lifestyle
choices.

# Our Solution
Carbon Quest allows users to calculate their personal carbon footprint, visualize their emissions using interactive charts, receive personalized suggestions on
how to reduce emissions, and track their progress over time. Gamification elements such as points, badges, rewards, and a leaderboard encourage users
to adopt sustainable habits through positive reinforcement.

# Target Users
- Students and young adults  
- Environmentally conscious individuals  
- Educational institutions and eco-clubs
  
# Key Features
- Personal carbon footprint calculator  
- Interactive pie chart visualization  
- Reduction tips based on user activity  
- Carbon points and rewards system  
- Achievement badges  
- Leaderboard for healthy competition
